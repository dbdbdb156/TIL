# Java
- java 에서는 JVM 운영체제의 역할을 한다.
- 자바에는 프로세스가 존재하지 않고 스레드에만 존재, 자바 스레드는 JVM 에 의해 스케쥴되는 실행단위 코드 블록이다.
- 자바에서 스레드 스케줄링은 전적으로 JVM에 의해 이루어진다.
* JVM이 관리하는 것 ?
 1) 스레드가 몇 개 존재하는지
 2) 스레드로 실행되는 프로그램 코드의 메모리 위치는 어디인지
 3) 스레드 상태는 무엇인지
 4) 스레드 우선순위는 얼마인지
-> 개발자는 자바 스레드로 작동할 스레드 코드를 작성하고, 스레드 코드가 생명을 가지고 실행을 시작하도록 JVM에 요청하는 일뿐이다.
