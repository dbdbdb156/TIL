# 알고리즘
## 이분탐색(,파라메트릭 서치) & 연결리스트
### 개념 공부

1. 연결리스트
1) ArrayList 와 LinkedList 차이
* ArrayList : 배열을 이용해서 List 인터페이스 구현
특징)
- index 를 통한 데이터 검색에 용이
: 검색이 많을때 자료구조 선택
시간복잡도)
- 검색 : O(n)
- 삽입삭제 : O(1)
: 이전 노드와 다음 노드를 참조하는 상태만 변경하면 됨.

* LinkedList : 각각의 노드를 연결하여 List 인터페이스 구현, 양방향 연결리스트임
특징)
- 데이터를 추가하거나 삭제하는 것이 원활
: 삽입,삭제가 잦을때 사용
함수)
- indexOf() : 값이 존재하는지 여부와 존재 시 위치를 알 수 있음.
시간복잡도)
- 검색 : O(n)
- 삽입,삭제 : O(n)
: 삽입, 삭제 이후 다른 데이터를 복사해야 하기 때문에, 한가지를 삭제하면 복사하여 앞으로 한칸씩 당겨서 저장해야함.

---------------

2. 이분탐색과 파라메스틱서치 차이
1) 이분탐색 : 원하는 값을 찾는과정
: 즉 일치하는지를 확인한다.
2) 파라메트릭 서치 : 값들의 점검이 연속적일때, 범위를 통해서 해당되는 값들을 찾아감
즉, 주어진 범위 내에서 원하는 값 또는 원하는 조건에 가장 일치하는 값을 찾아내는 알고리즘
ex) false, true 가 sorting 된 경우, 숫자가 연속적일 경우
