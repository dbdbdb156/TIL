# 알고리즘
## DFS, TREE
### 백준

* 요지
- Tree 를 루트 없는 그래프로서 볼수 있어야 한다
- 끝 노드를 찾기 위해서는 트리에 간선에 가중치가 있을때 dfs 를 통해서 가장 가중치가 긴 노드를 통해서 위치를 찾고
가장 끝 노드에서 dfs 를 다시 돌리면 가장 가중치가 큰 위치에 값을 구한다.
** 즉, dfs 를 2번 돌리므로서 찾을수 있음.
-> 어떤 노드에서든 가장 가중치 큰 노드를 찾아서 그 노드에서 dfs 를 돌리면 큰 값을 가질수 있음.

1. 1167 트리의 지름
- 풀이 : dfs 를 돌리며 dfs 함수의 스택 한 상태에서 그 값이 최대값인지를 계속해서 확인하며 search 후 값 최신화
- 애로사항 : if 문 안에 return 을 걸어버리면 dfs 안쪽으로 들어가면서의 값밖에 구해지지 않는다. if 문을 실행후 즉 max 값을 확인후 dfs 를 그대로 실행해야한다. return 하면 안된다.

https://www.acmicpc.net/problem/1167
