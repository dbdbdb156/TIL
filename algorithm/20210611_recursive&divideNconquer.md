#### 2. 분할 정복
: 문제를 거의 같은 크기로 나눈 부분문제로 나눠서 푸는것이 핵심
* 핵심요소
1) 문제를 더 작은 문제로 분할하는 과정
2) 각 문제에 대해 구한 답을 원래 문제에 대한 답으로 병합하는 과정
3) 답을 분할하지 않고 곧장 풀수 있는 매우 작은 문제

ex) 행렬의 거듭제곱
-참고 : 백준 10830 행렬제곱
https://www.acmicpc.net/problem/10830

ex) 병합정렬과 퀵정렬의 가장 큰 차이.
시간 걸리는 작업을 분할 단계에서 하느냐, 병합 단계에서 하느냐의 차이이다.

ex) 카라츠바의 빠른 곱셈 알고리즘
- (ax+b)*(cx+d) = acx^2+bc+ad*x+bd = z1*x^2+z2*x+z3 의 형태로 4번의 곱셈이 아닌 3번의 곱셈만 할수 있게 수정하는것임.
