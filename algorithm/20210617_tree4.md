# 알고리즘
## 트리,그래프,DFS
### 백준

1. 13016 내 왼손에는 흑염룡이 잠들어 있다
- 풀이 : dfs 를 이용해서 가장 긴 길이의 node 를 찾아 끝노드 n1,n2 를 찾은후, 그 노드에서 dfs 를 하며 각 노드의 값을 찾아
배열에 관리된 값들의 max 값을 도출
- 근거 :
1) 전형적인 트리를 Graph 로 변형하여 dfs 를 통해 트리 구현
2) n = 50000 이기에 O(n^2) 이면 시간 초과가 남
3) 가장 긴 길이의 노드 두개를 dfs 를 통해서 찾을수 있음.
- 애로사항
1) dfs를 O(n^2) 번을 돌리니 시간초과가 남
2) dfs 를 재귀함수가 원하는 값을 return 하는 형식으로 짰을떄는 원하는 끝 node 를 찾을수 없음.
3) dfs 함수 안에 변수들을 넣고, 최대값을 찾는 과정을 잘 조절해야함.
