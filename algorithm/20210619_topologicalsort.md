# 알고리즘
## 위상정렬 및 그래프&트리
### 백준

1. 21276 계보 복원가 호석
- 풀이 : 트리구조를 구현하는데에는 그래프로서 bfs를 한다면 트리가 구현 된다.
위상 정렬을 통해서 부모와 자식간의 관계를 파악한후 출력한다.
- 근거 :
1) 부모와 자식간의 계층관계가 있기에, 위상정렬을 사용한다.
+ 위상정렬을 나타날때, indegree[] 배열을 선언하여 자식 노드가 얼마나 선택이 되었는지, 선택이 되지 않으면 부모노드임을 확인할수있음.
2) 문자열 형태의 트리를 만들어야 하기에, 문자열을 인덱스로 변환해줌. 즉, Map 을 사용해서, 인덱스로서 관리를 하고 출력함.

* 위상정렬을 나타낼때는 indegree 가 0일때 q에 쌓아서 확인을 하는 형태이므로, bfs 를 통해서 큐의 값을 확인하는것이 좋다.
