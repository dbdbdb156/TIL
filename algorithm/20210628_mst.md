# 알고리즘
## 최소신장트리
### 백준

1. 1197 최소 스패닝 트리
- 풀이 : 유니온 파인드 자료구조를 통한 집합 설계. 크루스칼 알고리즘 사용. 우선순위큐로 weight 순으로 데이터를 관리.
두개의 노드가 같은 부모를 가지고 있지 않다면 집합을 연결해주는 형태
- 근거 : 
1) 크루스칼 알고리즘은 가중치가 작은 순서대로 값들의 모음을 관리하기에 집합이 적합함
2) 집합을 나타내는데는 유니온 파인드가 적합함.
3) 가중치 순으로 우선순위큐를 관리하는것이 크루스칼 알고리즘에서 필요함.

https://www.acmicpc.net/problem/1197

2. 14621 나만 안되는 연애
- 풀이 : 최소신장트리, 위와 동일, 차이는 문제에서 M,W 의 집합일때만 가중치를 선택함.

https://www.acmicpc.net/problem/14621

3. 1647 도시 분할 계획
- 풀이 : 최소신장트리를 구현한후 n-2 개까지만 크루스칼 알고리즘을 적용한후 출력
- 근거 : 두개의 집합으로 나누기 위해서는, 노드 n 개일때, 가중치를 n-1개 선택한것에서 가장 큰 한 가중치를 제외하면 가장 작은 분할된 마을이 된다.
- 애로사항 :
1) 강결합 컴포넌트인줄 알았음. 문제가 헷갈려서 몇번이나 다시 읽을 필요가 있었음.
2) 강결합 컴포넌트 정리해보기

https://www.acmicpc.net/problem/1647
