# 알고리즘
## 구현
### 프로그래머스

1. 메뉴 리뉴얼
- 풀이 : map 과 리스트를 통한 구현. 최대 갯수를 구하기 위한 배열을 관리하는것이 핵심.

https://programmers.co.kr/learn/courses/30/lessons/72411

2. 순위 검색
- 풀이 : 재귀 함수를 통한 상태 관리와 이분탐색을 통한 범위 최대갯수 구하기. 배열을 상태를 관리하기위한 메모라이제이션으로 사용.

https://programmers.co.kr/learn/courses/30/lessons/72412

3. 합승 택시 요금
- 풀이 : 다익스트라 3번을 통한 각자 길이의 최소값을 구하는것이 핵심
- 애로사항 : int 의 최대값은 21억 정도 되기에, INF 값을 9억으로 두면 오버플로우가 일어난다.

https://programmers.co.kr/learn/courses/30/lessons/72413
