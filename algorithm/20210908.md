# 알고리즘
## 위상정렬 & 이분탐색 & 구현 & DP 
### 백준

1. 9465 스티커
- 풀이 : DP를 통해서 dfs 구현

https://www.acmicpc.net/problem/9465

2. 1913 달팽이
- 풀이 : 구현, 반복문을 돌리며 제어 값이 나오면 방향 수정

https://www.acmicpc.net/problem/1913

3. 1939 중량제한
- 풀이 : 그래프 탐색(bfs or dfs) 를 통해서 해당 값이 통과 가능한지를 파악.
값을 찾는 방법은 이분탐색

https://www.acmicpc.net/problem/1939

4. 2110 공유기 설치
- 풀이 : 이분탐색을 통해서 공유기의 값이 가능한지 점검
- 애로사항 : 답은 나왔지만, 풀이방향을 이해하지 못함. 핵심은 해당 값이 작거나 같으면 해당 노드에 공유기를 설치해버리는것이 핵심.

https://www.acmicpc.net/problem/2110

5. 21276 계보 복원가 호석
- 풀이 : 위상정렬, 부모와 자식의 관계성을 찾을때, 조부모와 자식의 관계성을 찾을때에도 위상정렬을 찾을수 있다.
즉, 계층의 차이가 있을때 위상정렬을 사용할수 있음을 명시

https://www.acmicpc.net/problem/21276
