# 알고리즘
## DFS & 트리DP & BFS & 트리
### 백준

1. 1167 트리의 지름
- 풀이 : 트리의 지름 구하는 문제. 한 노드에서 dfs 후 길이가 가장 큰 idx 에서 다시 dfs 를 하여 지름을 구함.

https://www.acmicpc.net/problem/1167

2. 1520 내리막 길
- 풀이 : dp 문제. bfs를 통해 원하는 값에 도달했을시 값을 추가하며 dp[0][0]의 값을 저장.
- dp의 의미 : 도달 했을때에 도달한 방법의 횟수를 저장하는 배열

https://www.acmicpc.net/problem/1520

3. 17831 대기업 승범이네
- 풀이 : 트리 DP 문제. index 의 값은 현재의 노드, state 의 값은 1은 부모노드일때, 0은 자식노드일때
1) state = 1 일때 ( 부모노드일때 ), 자식들이 모두 부모라고 가정을 한다.  정확히 말하면, state =1 에서 state = 0 까지 가야지 부모 자식 관계가 설정이 되는 것이고
 state = 1만 있을 경우는 아무것도 연결되지 않은 선택되지 않은 상황이다.
2) state = 1 중에, 한개는 부모 자식간의 관계, 나머지는 부모 자식간의 관계가 아닐 경우를 모두 계산.

https://www.acmicpc.net/problem/17831

4. 1949 우수마을
- 풀이 : 트리 DP 문제. dfs 를 통해서 탐색하며, 각 노드에서 가장 큰 값을 저장 함.
1) state = 1은 우수마을일때, 무조건 옆마을은 우수마을이 아님
2) state = 0은 우수마을이 아니면, 옆마을은 우수마을일수도, 아닐수도 있다.

5. 9019 DSLR
- 풀이 : bfs 문제. 배열의 수가 한정적이고, 다른 수를 변환시켜서 원하는 값을 찾는 경우에, bfs 를 설계함으로서 원하는 값을 구할수 있다.

https://www.acmicpc.net/problem/9019
