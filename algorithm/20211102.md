# 알고리즘
## 피보나치 수열 & 피사노주기 & 분할정복
### 백준

1. 피보나치 수 3
- 풀이 :
1) 피보나치 점화식을 통한 행렬 생성 및 재귀함수를 통한 분할정복으로 O(logN) 의 시간 복잡도로 행렬 곱셈 계산
피보나치 점화식 : F(i+1) = F(i)+F(i-1) => 	( F(i+1) )  = | 1 1 | ( F(i)   )
		// 				F(i)   = F(i) + 0			( F(i)	 )	  | 1 0 | ( F(i-1) )
		//	( F(i+1) )  = | 1 1 | ( F(i)   ) => ( F(i+1) )  = | 1 1 |^i ( F(1) )
		// 	( F(i)	 )	  | 1 0 | ( F(i-1) ) => ( F(i)	 )	  | 1 0 | 	( F(0) )
		// F(1) = 1, F(0) = 0
		//	( F(i+1) )  = | 1 1 |^i ( F(1) ) => ( F(i+1) )  = | 1 1 |^i ( 1 )
		// 	( F(i)	 )	  | 1 0 | 	( F(0) ) => ( F(i)	 )	  | 1 0 | 	( 0 )
2) 피사노주기 : 피보나치 수열도 주기를 가지고 있음.
나누는 수가 k=10^n 일때, 피사노주기(피보나치 수열 주기) = 15*10^(n-1)
