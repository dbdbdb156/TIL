# 반응형 웹

: pc, 스마트기기, 내비게이션 등 기기 또는 환경마다 최적화된 웹사이트를 제공해주는것

#### 검색엔진 최적화 
포털 사이트에서 특정 키워드로 저장 했을때, 검색 결과에서 상위권 나타나도록 관리하는 작업
; 주소를 똑같은걸 쓰기때문에 

## 핵심내용
#### 가변그리드
: 고정적인 값(px)이 아닌 화면의 크기에 따라 가변적인 퍼센트를 사용하는 기술
but, 기기나 환경에 따라 구조를 바꾸지는 못함. 그래서 화면을 제어할 뷰포트,, 화면의 크기나환경을 감지해서 구조를 바꿀 미디어 쿼리가 필요

#### 미디어 쿼리
: 기기에게 미디어 종류와 화면 크기를 파악 후, 웹사이트 형태를 변경하는 기술

#### 뷰포트
: 화면에 보이는 영역을 제어하는 기술, 스마트 기기의 보이는 영역을 기기의 실제 화면 크기로 변경하여
미디어 쿼리가 화면의 크기를 정확하게 감지 할수 잇도록 하기 위함
ex) 데스크 톱은 해상도에 맞춰서, 스마트 폰은 기본 설정값이 자동으로 보이는 영역이기에 정확한 감지 어려움

#### 플렉서블 박스
: 가변적인 박스를 만드는 기술, 웹사이트 구조 설계를 위한 기술

### 잔도
내용 : 0 ~ 44
느낀점 : 웹을 다루는 데에 있어 각광을 받는 반응형 웹을 간단하게 배워보는 기회인것 같다.
오랜만에 html의 예제를 다루며, 예전 DB 수업을 들을때 프로젝트 했던 웹포스기 만드는데에 코드를 이해하기 위해노력했던 것들이,
도움이 된것 같고, 빠르게 속독하고 웹을 다루는 기술들을 배우는 기회가 되도록 해야겠다.
